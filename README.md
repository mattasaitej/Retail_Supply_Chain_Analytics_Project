# 📦 Retail Supply Chain Analytics Project

## 📌 Project Overview

This project presents an end-to-end retail supply chain analytics solution using **Python**, **Jupyter Notebook**, and **Power BI**. The objective is to transform raw retail supply chain data into actionable business insights through data cleaning, exploratory data analysis, and interactive visualizations.

---

## 🎯 Project Objectives

* Clean and preprocess raw supply chain data.
* Perform exploratory data analysis (EDA).
* Analyze sales, revenue, and profitability trends.
* Evaluate the impact of discounts on profit.
* Identify top-performing products and customer segments.
* Analyze logistics and shipping performance.
* Visualize geographic sales distribution.
* Develop an interactive Power BI dashboard for business decision-making.
